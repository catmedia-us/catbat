Environment Variables

The following environment variables are used and should be defined when using this project

-----------
Development
-----------
%CURRENT_WORKSPACE%
This points to the current (Eclipse) Workspace and parent folder of this project.

-----------
Runtime
-----------
%BatchPath%
The folder to hold Scripts and Batch files like VBXCopy
